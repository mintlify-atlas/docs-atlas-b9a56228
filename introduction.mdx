## Welcome to the Asterisk PBX Tutorial

This tutorial was developed to provide comprehensive guidance on installing and configuring **[Asterisk](https://www.asterisk.org/)**, a powerful open-source Private Branch Exchange (PBX) system. Originally created as a university project, it has been expanded into a full tutorial to help others navigate the configuration of Asterisk.

<Note>
  This is a first-time configuration guide for a [PBX](https://es.wikipedia.org/wiki/PBX) system like Asterisk. While there may be multiple ways to accomplish these configurations, this tutorial provides a solid foundation for getting started.
</Note>

## Tutorial Approach

The tutorial is organized into small, modular sections to make it easier to:
- Understand core concepts
- Test integrated components
- Build towards a complete [Call Center](#7-emulación-de-un-call-center-de-un-centro-de-salud) implementation

Each module introduces specific functionality that can be tested independently before moving to more complex configurations.

## What You'll Learn

This tutorial covers a wide range of Asterisk features and configurations:

- **Basic Setup**: SSH connection, installation, and initial configuration
- **PBX Functionalities**: 
  - Voicemail (buzón de voz)
  - Call forwarding and transfer
  - SIP messaging
  - Call distribution and pickup
  - Text-to-Speech with Festival
  - Call security
  - Business hours configuration
  - Logging and monitoring
  - Queue management
- **Database Integration**: MariaDB integration with Asterisk
- **PBX Interconnection**: Using the DUNDi module
- **Real-World Application**: Call Center emulation for a health center
