This example demonstrates a comprehensive call center configuration for a health center (centro de salud) with multiple departments, call queues, database integration, voicemail, and Festival text-to-speech.

## Overview

This advanced setup includes:
- **Call queues** for registered and non-registered patients
- **ODBC database integration** for user lookups
- **Department-based routing** (Reception, Management, Doctors, Nurses)
- **IVR menu system** with business hours checking
- **Voicemail** with PIN authentication
- **Call pickup groups** by department
- **Festival TTS** for voice prompts
- **Message center** for SIP messaging
- **DUNDi** for multi-PBX interconnection

## Architecture

The call center routes incoming calls through:

1. **Time-based routing**: Checks business hours (Mon-Fri, 8:00-20:00)
2. **IVR menu**: Registered vs non-registered users
3. **User authentication**: Database lookup for registered users
4. **Queue distribution**: Routes to appropriate agent queue
5. **Department contexts**: Different permissions by role

```ini /etc/asterisk/extensions.conf
[general]
static=yes
writeprotect=no
clearglobalvars=no

[public]
include => exterior

;####-INI-LLAMADAS-####
[exterior]
; Redirect to dialed extension with business hours (Mon-Fri 8:00-20:00)
exten => 757112233,1,GotoIfTime(8:00-20:00,mon-fri,*,*?abierto,1,1)
same => n,Festival(En estos momentos no hay nadie que te pueda atender)
same => n,Festival(Nuestro horario es de 8 a 20 horas de lunes a viernes)
same => n,Hangup()

[abierto]
; The health center is open
exten => 1,1,Answer()
same => n,Festival(Bienvenido al centro de salud)
same => n(loop),Festival(Presione)
same => n,SayNumber(1)
same => n,Festival(si es un usuario registrado o)
same => n,SayNumber(2)
same => n,Festival(en caso contrario)
same => n,Read(opcion,,1,3)           ; Read 1 digit, 3 attempts
same => n,SayNumber(${opcion})
same => n,GotoIf($["${opcion}" = "1"]?10,1)
same => n,GotoIf($["${opcion}" = "2"]?20,1)
same => n,Festival(Opcion incorrecta)
same => n,Goto(1,loop)
same => n,Hangup()

; Registered user flow
exten => 10,1(id),Festival(Por favor marque su numero de identificacion o presione)
same => n,SayNumber(9)
same => n,Festival(para volver al menu)
same => n,Read(id,,4,3,5)            ; Read 4-digit ID
same => n,SayNumber(${id})
same => n,GotoIf($["${id}" = "9"]?1,loop)
same => n,Set(nombre=${ODBC_usuario(${id})})  ; Database lookup
same => n,GotoIf($["${nombre}" != ""]?recepcion,1,1)
same => n,Festival(Usario no encontrado)
same => n,Goto(10,id)
same => n,Hangup()

; Non-registered user flow
exten => 20,1,Goto(recepcion,2,1)
same => n,Hangup()

[recepcion]
; Call to group
exten => _X00,1,Festival(Llamada a grupo ${EXTEN})
same => n,Goto(llamar-grupo,${EXTEN},1)
same => n,Hangup()

; Registered users queue
exten => 1,1,Set(CHANNEL(musicclass)=recepcion)
same => n,Answer()
same => n,Festival(Hola ${nombre})
same => n(reg),Festival(Por favor espere a que le atiendan)
same => n,Queue(registrados,,,,60)   ; Queue with 60 sec timeout
same => n,Goto(1,reg)
same => n,Hangup()

; Non-registered users queue
exten => 2,1,Set(CHANNEL(musicclass)=recepcion)
same => n,Answer()
same => n(noreg),Festival(Por favor espere a que le atiendan)
same => n,Queue(noregistrados,,,,60)
same => n,Goto(2,noreg)
same => n,Hangup()

; Call to another extension
exten => _XXX,1,Set(extension=${EXTEN})
same => n,Set(nombre=${ODBC_extension(${extension})})
same => n,GotoIf($["${nombre}" != ""]?llamar,${extension},1)
same => n,Festival(La extension ${extension} no se encuentra disponible)
same => n,Hangup()

; Voicemail access
exten => _*XXX,1,Set(llamante=${CALLERID(num)})
same => n,Goto(buzon-voz,1,1)
same => n,Hangup()

; Call pickup
exten =>  _*8.,1,Set(extension=${EXTEN:2})
same => n,Set(pref=${extension:0:1})
same => n,GotoIf($["${pref}" = "1"]?llamar,8,1)
same => n,Festival(No tiene permiso para capturar una llamada a la extension ${extension})
same => n,Hangup()

; External calls
exten => _XXXXXXXXX,1,Goto(llamar,${EXTEN},1))
same => n,Hangup()

; Unrecognized number
exten => _X!,1,Festival(El telefono o extension)
same => n,SayNumber(${EXTEN})
same => n,Festival(es incorrecto)
same => n,Hangup()

[direccion]
; Management department context
exten => _X00,1,Festival(Llamada a grupo ${EXTEN})
same => n,Goto(llamar-grupo,${EXTEN},1)
same => n,Hangup()

exten => _XXX,1,Set(extension=${EXTEN})
same => n,Set(nombre=${ODBC_extension(${extension})})
same => n,GotoIf($["${nombre}" != ""]?llamar,${extension},1)
same => n,Festival(La extension ${extension} no se encuentra disponible)
same => n,Hangup()

exten => _*XXX,1,Set(llamante=${CALLERID(num)})
same => n,Goto(buzon-voz,1,1)
same => n,Hangup()

; Management can pickup any call
exten =>  _*8.,1,Set(extension=${EXTEN:2})
same => n,Goto(llamar,8,1)
same => n,Hangup()

exten => _XXXXXXXXX,1,Goto(llamar,${EXTEN},1))
same => n,Hangup()

exten => _X!,1,Festival(El telefono o extension)
same => n,SayNumber(${EXTEN})
same => n,Festival(es incorrecto)
same => n,Hangup()

[doctores]
; Doctors department context
exten => _X00,1,Festival(Llamada a grupo ${EXTEN})
same => n,Goto(llamar-grupo,${EXTEN},1)
same => n,Hangup()

exten => _XXX,1,Set(extension=${EXTEN})
same => n,Set(nombre=${ODBC_extension(${extension})})
same => n,GotoIf($["${nombre}" != ""]?llamar,${extension},1)
same => n,Festival(La extension ${extension} no se encuentra disponible)
same => n,Hangup()

exten => _*XXX,1,Set(llamante=${CALLERID(num)})
same => n,Goto(buzon-voz,1,1)
same => n,Hangup()

; Doctors can only pickup calls to 3XX extensions
exten =>  _*8.,1,Set(extension=${EXTEN:2})
same => n,Set(pref=${extension:0:1})
same => n,GotoIf($["${pref}" = "3"]?llamar,8,1)
same => n,Festival(No tiene permiso para capturar una llamada a la extension ${extension})
same => n,Hangup()

exten => _XXXXXXXXX,1,Goto(llamar,${EXTEN},1))
same => n,Hangup()

[enfermeria]
; Nurses department context
exten => _X00,1,Festival(Llamada a grupo ${EXTEN})
same => n,Goto(llamar-grupo,${EXTEN},1)
same => n,Hangup()

exten => _XXX,1,Set(extension=${EXTEN})
same => n,Set(nombre=${ODBC_extension(${extension})})
same => n,GotoIf($["${nombre}" != ""]?llamar,${extension},1)
same => n,Festival(La extension ${extension} no se encuentra disponible)

exten => _*XXX,1,Set(llamante=${CALLERID(num)})
same => n,Goto(buzon-voz,1,1)
same => n,Hangup()

exten =>  _*8.,1,Set(extension=${EXTEN:2})

same => n,GotoIf($["${pref}" = "4"]?llamar,8,1)
same => n,Festival(No tiene permiso para capturar una llamada a la extension ${extension})
same => n,Hangup()

exten => _XXXXXXXXX,1,Goto(llamar,${EXTEN},1))
same => n,Hangup()

[llamar]
; Execute call to extension
exten => _XXX,1,Festival(Llamando a ${nombre})
same => n,Dial(SIP/${extension},15)
same => n,VoiceMail(${extension})    ; If no answer, go to voicemail
same => n,Hangup()

; Call pickup execution
exten => 8,1,Festival(Capturando llamada a la extension ${extension})
same => n,Set(equipo=${ODBC_equipo(${extension})})
same => n,Pickup(${extension}@${equipo})
same => n,Hangup()

; DUNDi connection for external PBX
exten => _XXXXXXXXX,1,Macro(dundi-priv,${EXTEN:5})
same => n,Hangup()

[llamar-grupo]
; Group calling
exten => 100,1,Dial(SIP/101&SIP/102&SIP/103&SIP/110,60)  ; Reception group
same => n,Hangup()

exten => 200,1,Dial(SIP/201&SIP/202&SIP/203,60)          ; Management group
same => n,Hangup()

exten => 300,1,Dial(SIP/301&SIP/302&SIP/303,60)          ; Doctors group
same => n,Hangup()

exten => 400,1,Dial(SIP/401&SIP/402&SIP/403,60)          ; Nurses group
same => n,Hangup()

exten => s,1,Festival(Grupo ${extension} no se encuentra disponible)
same => n,Hangup()

[buzon-voz]
; Voicemail access with PIN
exten => 1,1,Festival(Para acceder a su buzon de voz)
same => n(pin),Festival(Marque su numero de identificacion privado)
same => n,Read(pin,,4,3,5)
same => n,Set(idpin=${ODBC_pin(${llamante})})  ; Get PIN from database
same => n,GotoIf($["${pin}" = "${idpin}"]?2,1)
same => n,Festival(Numero de identificacion erroneo)
same => n,Goto(1,pin)
same => n,Hangup()

exten => 2,1,Festival(Marque 1 para continuar)
same => n,VoiceMailMain(${llamante})
same => n,Hangup()
;####-FIN-LLAMADAS-####

;####-INI-MENSAJES-####
[suscribir]
; Extension hints for presence
exten => 101,hint,SIP/101
exten => 102,hint,SIP/102
exten => 103,hint,SIP/103
exten => 110,hint,SIP/110
exten => 201,hint,SIP/201
exten => 202,hint,SIP/202
exten => 203,hint,SIP/203
exten => 301,hint,SIP/301
exten => 302,hint,SIP/302
exten => 303,hint,SIP/303
exten => 401,hint,SIP/401
exten => 402,hint,SIP/402
exten => 403,hint,SIP/403

[mensajes]
; SIP message routing
exten => _X.,1,Set(dest=${EXTEN})
same => n,Set(remitente=${CUT(MESSAGE(from),<,2)})
same => n,Set(remitente=${CUT(remitente,@,1)})
same => n,Set(remitente=${CUT(remitente,:,2)})
same => n,Set(texto=${MESSAGE(body)})
same => n,Set(MESSAGE(body)=${remitente}: ${texto})
same => n,Set(nombre=${ODBC_extension(${remitente})})
same => n,Set(nombre=${IF($[ "${nombre}" = ""]?desconocido)})
same => n,MessageSend(sip:${dest}, Centro de Mensajes(CdM))
same => n,Noop(Estado del mensaje ${MESSAGE_SEND_STATUS})
same => n,GotoIf($["${MESSAGE_SEND_STATUS}" != "SUCCESS"]?fallo,s,1)
same => n,Hangup()

[fallo]
exten => s,1,Set(MESSAGE(body)=CdM: El mensaje "${texto}" para ${dest} no ha sido enviado)
same => n,Set(remit=${CUT(MESSAGE(from),<,2)})
same => n,Set(remit=${CUT(remit,@,1)})
same => n,MessageSend(${remit},Centro de Mensajes(CdM))
same => n,NoOp(Estado del mensaje ${MESSAGE_SEND_STATUS})
same => n,Hangup()
;####-FIN-MENSAJES-####

;####-INI-DUNDI-####

exten => _XXXXXXXXX,1,Dial(Zap/g1/${EXTEN:6},20,rtT)
```

## Database Schema

The call center requires a MySQL database with these tables:

```sql
CREATE DATABASE asterisk;

USE asterisk;

-- Users table (patients)
CREATE TABLE usuarios (
    id VARCHAR(10) PRIMARY KEY,
    nombre VARCHAR(100) NOT NULL,
    extension VARCHAR(10)
);

-- Personnel table (staff)
CREATE TABLE personal (
    extension VARCHAR(10) PRIMARY KEY,
    nombre VARCHAR(100) NOT NULL,
    id VARCHAR(10) NOT NULL,
    equipo VARCHAR(20) NOT NULL  -- Department: recepcion, direccion, doctores, enfermeria
);

-- Sample data
```

## Queue Configuration

Create `/etc/asterisk/queues.conf`:

```ini
[general]
persistentmembers = yes
monitor-type = MixMonitor

[registrados]
strategy = leastrecent      ; Route to agent who least recently received a call
timeout = 30                 ; Ring agent for 30 seconds
music = recepcion           ; Music on hold class
member => SIP/101
member => SIP/102
member => SIP/103

[noregistrados]
strategy = leastrecent
timeout = 30
music = recepcion
member => SIP/110
```

## Key Features Explained

### Time-Based Routing

The `GotoIfTime()` application checks business hours:

```
exten => 757112233,1,GotoIfTime(8:00-20:00,mon-fri,*,*?abierto,1,1)
```

### Database Integration

ODBC functions retrieve user data:

```
Set(nombre=${ODBC_usuario(${id})})
```

### Call Pickup Permissions

Different departments have different pickup permissions:
- **Reception**: Can pickup 1XX extensions only
- **Management**: Can pickup any extension
- **Doctors**: Can pickup 3XX extensions only
- **Nurses**: Can pickup 4XX extensions only

### Queue Strategy

`leastrecent` strategy distributes calls to the agent who least recently received a call, ensuring fair distribution.

## Troubleshooting

### Database Connection Issues

Test ODBC connection:

```bash
isql -v asterisk username password
```

### Festival Not Working

Check Festival is running:

```bash
sudo systemctl status festival
telnet localhost 1314
```

### Queue Not Working

Verify queue members:

```
asterisk> queue show registrados
```

Add members dynamically:

```
asterisk> queue add member SIP/101 to registrados
```
